Memory 3: Echoing Winds

Meningkatkan 15% Damage Wind Deflect. Meningkatkan Level Wind Deflect sebanyak +3. Maksimal hingga Level 15..

Memory 4: Crosswinds

Memory 5: Flowing Winds

Meningkatkan 5% Movement Speed. Meningkatkan Level Random Winds sebanyak +3. Maksimal hingga Level 15..

Memory 6: Storm Winds

Saat menggunakan Random Winds, seluruh musuh di radius akan ditarik ke pusat. Setelah Nana menggunakan Random Winds, akan muncul ledakan angin yang akan melempar musuh ke udara dan memberikan Wind Damage AOE sebesar 200% ATK Nana..
